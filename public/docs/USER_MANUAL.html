❓ 常见问题

Q1: 如何添加新的数据源？

路径: Tower → 数据源管理
1. 点击「添加数据源」
2. 选择适配器类型
3. 配置 API Key（如有）
4. 测试连接
5. 启用

Q2: 评分规则如何生效？

新发现的候选: 立即应用新规则
已有候选: 下次合格化时应用
手动重新评分: 知识引擎 → 评分规则 → 点击「重新评分所有候选」

Q3: 如何查看新兴市场客户？

路径: 获客雷达 → 候选客户
筛选: 地区（MENA / AFRICA / LATAM / ECA）
数据源: emerging_markets / dev_bank
